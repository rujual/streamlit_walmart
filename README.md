### Steps to Run

Clone/Download- git clone https://github.com/rujual/streamlit_walmart.git

Install Streamlit Python library- 
''' console
$ pip install streamlit
'''

Open terminal and Go into downloaded directory-

Run - 
''' console
$ streamlit run mall_customer.py
'''
